# Enneagram Personality Test (Myanmar) 🇲🇲

Welcome to the **Enneagram Personality Test**! This project is a simple, fast, and user-friendly single-page web application that allows users to discover their true Enneagram personality type through a 144-question test in the Myanmar (Burmese) language.

## 🌟 Features
* **144 Comprehensive Questions:** Carefully structured and formatted for native Myanmar speakers to easily understand and answer.
* **Instant Results:** Automatically calculates the user's responses and reveals their primary personality type along with the top 3 traits immediately after submission.
* **Client-Side Only:** Fully runs on the browser using pure HTML, CSS, and JavaScript. No database or server setup is required.
* **Responsive Design:** A clean, minimalistic, and easy-to-read interface suitable for both mobile phones and desktop computers.

## 🚀 How to Take the Test
1. Visit the live website here: **[https://mgogga-mm.github.io/mgogga-enneagram-test/]**
2. Read each question carefully and select the option that best describes you.
3. Scroll to the bottom and click the **"ရလဒ် အဖြေကို တွက်ချက်မည်"** (Calculate Result) button.
4. Instantly discover your dominant Enneagram personality type!

## 🛠️ Built With
* **HTML5** - For the structure of the web page and question forms.
* **CSS3** - For styling, layout, and providing a clean user experience.
* **JavaScript** - For scoring logic and instant result generation.
